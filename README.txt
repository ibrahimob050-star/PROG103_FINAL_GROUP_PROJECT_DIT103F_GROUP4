STUDENT ACADEMIC PORTAL - WORKING VERSION

How to run:
1. Make sure Python is installed.
2. Open terminal/cmd in this folder.
3. Run: python student_academic_portal_fixed.py

Features:
- Add student records
- Save records in student_data.json
- Prevent duplicate IDs
- Calculate grade and status automatically
- Search records
- Update selected record
- Delete selected record
- Generate summary report
- Clear all data

Grading:
A = 80-100 Excellent
B = 70-79 Good
C = 60-69 Credit
D = 50-59 Pass
F = 0-49 Fail
